Organogram Add-in for Excel
A custom Office Add-in that generates interactive organizational charts from Excel table data.

Features
Data-Driven: Generates charts automatically from a specified Excel table (OrgData).

Interactive: Drag-and-drop boxes to rearrange the layout.

Dynamic Lines: Connectors automatically redraw when boxes are moved.

Export: Export the chart to an image using html2canvas.

Installation (Side-loading)
To use this add-in, you must side-load it into Excel:

Open Excel on the web or desktop.

Go to the Insert tab and click My Add-ins.

Click the Manage My Add-ins dropdown and select Upload My Add-in.

Browse to select the manifest.xml file included in this repository.

Click Upload.

Usage
In Excel, click Create example table in the add-in pane to generate the required data structure.

Fill in your organization data.

Click Refresh to generate the visual organogram.
